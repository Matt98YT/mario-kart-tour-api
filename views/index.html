Mario Kart Tour Api

Rankings Brute Force Scraper

Mario Kart Tour is a great game, and has a global leaderboard. Unfortunately, they've made some great performance choices, and embedded the data that powers the leaderboard directly in the html page! This means that, for those of us that like data, it needs to be freed from the confines of the data attribute that it hides behind.

This project is about the simplest way possible to grab this data from the html using `cheerio`, and spit it out as a cors-accessible json endpoint.
